About


Map Link is a concept that allows readers to gain access to worldwide news in a graphical way. Our concept aims to improve the accessibility of news by using an interactive map which informs readers which location in the world is currently making the headlines. Based on our user research, we learned that not every person knows the location of the latest news. Thus, we decided to present news in a graphical way, which we believe will be able to improve user's geographical knowledge.

Anytime when a certain country has breaking news, the color of that country will be changed into a bright color. Therefore, users can easily recognise the location of the breaking news. users can simply click on the country that is changing in color, and a link will be provided to direct users to the actual news site where the article is based. In all, we hope that Map Link will be able to deliver the latest news and better understand the world.
